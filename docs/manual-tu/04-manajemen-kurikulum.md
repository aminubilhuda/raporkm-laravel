# Manajemen Kurikulum

Panduan mengelola mata pelajaran, tingkat, kompetensi, dan peta kelas.

## Mata Pelajaran

### Daftar Mapel

1. Klik menu **Mapel** di sidebar
2. Anda akan melihat daftar seluruh mata pelajaran
3. Gunakan **search** untuk mencari mapel

### Tambah Mapel

1. Klik **Tambah Mapel**
2. Isi data:
    - **Nama Mapel**: Contoh "Matematika"
    - **Kelompok Mapel**: A (Nasional), B (Kewilayahan), C (Kejuruan), D (Pilihan), E (Budi Pekerti)
    - **Urutan**: Nomor urut di rapor
3. Klik **Simpan**

### Edit/Hapus Mapel

- Klik ikon **Edit** untuk mengubah data
- Klik ikon **Trash** untuk menghapus

## Tingkat

1. Klik menu **Tingkat** di sidebar
2. Tambah tingkat (contoh: X, XI, XII)
3. Isi nama tingkat dan urutan

## Kompetensi Keahlian

1. Klik menu **Kompetensi** di sidebar
2. Tambah jurusan (contoh: TKJ, RPL)
3. Isi kode dan nama kompetensi

## Kelompok Mapel

1. Klik menu **Kelompok Mapel** di sidebar
2. Tambah kelompok (contoh: Muatan Nasional, Muatan Kejuruan)
3. Isi nama, kode, dan urutan

## Peta Kelas (Mapel Kelas)

### Assign Guru ke Mapel Kelas

1. Klik menu **Mapel Kelas** di sidebar
2. Pilih kelas
3. Untuk setiap mapel, pilih guru yang mengajar
4. Klik **Simpan**

!!! info "Penting"
    Assign guru ke mapel kelas sangat penting karena akan menentukan menu apa saja yang muncul di panel guru. Guru yang ditugaskan di mapel kelas akan melihat menu **Penilaian**, **Tujuan Pembelajaran**, dan **Lager Nilai**.

### Batch Update

1. Di halaman Mapel Kelas, ubah beberapa assign sekaligus
2. Klik **Simpan Semua** untuk update massal

## Mapel Siswa

1. Klik menu **Mapel Siswa** di sidebar
2. Pilih kelas
3. Assign mapel ke siswa (biasanya otomatis berdasarkan rombel)
4. Gunakan **Batch Update** untuk update massal
