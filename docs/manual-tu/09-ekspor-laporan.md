# Ekspor & Laporan

Panduan ekspor data ke Excel dan laporan pendidikan.

## Ekspor Nilai

1. Klik menu **Ekspor** di sidebar
2. Pilih **Ekspor Nilai**
3. Pilih kelas dan semester
4. Klik **Export Excel**
5. File Excel akan otomatis terdownload

## Ekspor Presensi

1. Klik menu **Ekspor** → **Ekspor Presensi**
2. Pilih kelas dan rentang tanggal
3. Klik **Export Excel**

## Ekspor Data Siswa

1. Klik menu **Ekspor** → **Ekspor Siswa**
2. Pilih kelas atau filter tertentu
3. Klik **Export Excel**

## Laporan Pendidikan

### Laporan Umum

1. Klik menu **Laporan** di sidebar
2. Anda akan melihat ringkasan data pendidikan
3. Data meliputi:
    - Jumlah siswa per kelas
    - Jumlah guru per mata pelajaran
    - Rekap nilai per kelas

### Laporan Kemendikbud

1. Klik **Laporan Pendidikan**
2. Data akan diformat sesuai format Kemendikbud
3. Klik **Export** untuk mengunduh

## Buku Induk

### Daftar Buku Induk

1. Klik menu **Buku Induk** di sidebar
2. Anda akan melihat daftar seluruh siswa

### Preview Buku Induk

1. Pilih siswa untuk melihat detail
2. Data meliputi:
    - Identitas siswa
    - Nilai per semester
    - Rekap presensi
    - Catatan wali kelas

### Cetak Buku Induk (PDF)

1. Klik **Cetak PDF**
2. File PDF buku induk akan terdownload

!!! info "Format Ekspor"
    Semua file ekspor menggunakan format `.xlsx` (Excel 2007+). Pastikan aplikasi yang Anda gunakan mendukung format ini.
