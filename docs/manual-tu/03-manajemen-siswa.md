# Manajemen Siswa

Panduan mengelola data siswa, rombongan belajar, dan anggota kelas.

## Daftar Siswa

1. Klik menu **Kesiswaan** di sidebar
2. Anda akan melihat daftar seluruh siswa
3. Gunakan **search** untuk mencari siswa
4. Gunakan **pagination** untuk navigasi

## Import Siswa dari Dapodik

1. Klik menu **Kesiswaan** → **Import**
2. Pilih file CSV/Excel yang sudah di-export dari Dapodik
3. Klik **Import**
4. Siswa akan otomatis terdaftar di sistem

!!! tip "Tips Import"
    - Pastikan format file sesuai template
    - NISN harus unik untuk setiap siswa
    - Data yang sudah ada tidak akan di-update ulang

## Tambah Siswa Manual

1. Klik **Tambah Siswa**
2. Isi data siswa:
    - **Nama Siswa**: Nama lengkap
    - **NIS**: Nomor Induk Siswa
    - **NISN**: Nomor Induk Siswa Nasional (unik)
    - **JK**: Jenis Kelamin (Laki-laki / Perempuan)
3. Klik **Simpan**

## Rombongan Belajar (Rombel)

### Daftar Rombel

1. Klik menu **Rombel** di sidebar
2. Anda akan melihat daftar seluruh rombel

### Tambah Rombel

1. Klik **Tambah Rombel**
2. Isi data:
    - **Nama Kelas**: Contoh "X TKJ 1"
    - **Tingkat**: Pilih tingkat (X, XI, XII)
    - **Kompetensi Keahlian**: Pilih jurusan
3. Klik **Simpan**

## Anggota Kelas

### Daftar Anggota Kelas

1. Klik menu **Anggota Kelas** di sidebar
2. Pilih kelas yang ingin dikelola

### Tambah Anggota Kelas

1. Pilih kelas terlebih dahulu
2. Klik **Tambah Anggota**
3. Pilih siswa yang akan dimasukkan ke kelas
4. Isi **Tahun Pelajaran** dan **Semester**
5. Klik **Simpan**

### Hapus Anggota Kelas

1. Di daftar anggota kelas, klik ikon **Trash** di baris yang ingin dihapus
2. Konfirmasi penghapusan

## Naik Kelas

1. Klik menu **Naik Kelas** di sidebar
2. Pilih kelas asal
3. Pilih kelas tujuan
4. Centang siswa yang akan naik kelas
5. Klik **Proses Naik Kelas**

!!! warning "Penting"
    Proses naik kelas hanya bisa dilakukan sekali per semester. Pastikan semua data sudah benar sebelum memproses.
