# Rapor & Cetak

Panduan mengelola deskripsi rapor dan cetak rapor digital.

## Deskripsi Rapor

### Daftar Deskripsi

1. Klik menu **Deskripsi Rapor** di sidebar
2. Anda akan melihat template deskripsi

### Tambah Deskripsi

1. Klik **Tambah Deskripsi**
2. Isi data:
    - **Jenis**: Deskripsi Umum / Deskripsi Per Kelas
    - **Teks**: Teks deskripsi yang akan muncul di rapor
3. Klik **Simpan**

### Edit/Hapus

- Klik ikon **Edit** untuk mengubah
- Klik ikon **Trash** untuk menghapus

## Preview Rapor

1. Klik menu **Rapor** di sidebar
2. Pilih siswa yang ingin di-preview
3. Pilih jenis rapor:
    - **Semester**: Rapor akhir semester
    - **Mid**: Rapor tengah semester
    - **PKL**: Rapor prakerin
4. Klik **Preview**

## Cetak Rapor (PDF)

### Cetak per Siswa

1. Pilih siswa
2. Klik **Cetak PDF**
3. File PDF akan otomatis terdownload

### Cetak Batch (ZIP)

1. Pilih beberapa siswa (centang checkbox)
2. Klik **Cetak Batch**
3. File ZIP akan terdownload berisi PDF per siswa

!!! tip "Tips Cetak"
    - Gunakan Chrome untuk hasil PDF terbaik
    - Pastikan printer sudah terkonfigurasi dengan benar
    - Untuk cetak batch, pastikan koneksi internet stabil

## Tanda Tangan Digital (Kepsek)

1. Login sebagai **Kepala Sekolah**
2. Buka menu **Rapor**
3. Pilih siswa
4. Rapor akan menampilkan tanda tangan digital kepsek
5. Klik **Cetak** untuk mencetak rapor yang sudah ditandatangani
