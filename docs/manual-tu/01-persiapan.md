# Persiapan Awal

Panduan login dan setup awal aplikasi E-Rapor KM.

## Login

1. Buka browser dan akses `https://km.smkabdinegara.sch.id`
2. Masukkan **username** dan **password** yang diberikan administrator
3. Klik **Log in**

!!! tip "Tips"
    - Gunakan browser Chrome atau Firefox versi terbaru
    - Simpan bookmark untuk akses cepat

## Setup Sekolah

Setelah login pertama kali, pastikan data sekolah sudah benar:

1. Klik menu **Pengaturan** di sidebar
2. Lengkapi data sekolah:
    - Nama sekolah
    - Alamat
    - NPSN / NSS
    - Logo sekolah
    - Favicon
3. Klik **Simpan**

## Setup Tahun Pelajaran

1. Buka menu **Pengaturan**
2. Di bagian **Tahun Pelajaran**, klik **Tambah**
3. Isi tahun pelajaran (contoh: `2025/2026`)
4. Set status menjadi **Aktif** untuk tahun yang sedang berjalan

## Setup Semester

1. Di bagian **Semester**, pilih semester yang aktif (Ganjil/Genap)
2. Klik **Set Semester** untuk mengaktifkan semester

!!! warning "Penting"
    Pastikan tahun pelajaran dan semester aktif sudah benar sebelum input nilai. Semua data nilai akan terkait dengan tahun pelajaran dan semester yang aktif.

## Menu Utama

Setelah login, Anda akan melihat sidebar dengan menu:

| Menu | Fungsi |
|------|--------|
| **Dashboard** | Ringkasan data sekolah |
| **Pegawai** | Kelola guru dan staf TU |
| **Kesiswaan** | Kelola data siswa |
| **Rombel** | Kelola rombongan belajar |
| **Mapel** | Kelola mata pelajaran |
| **P5BK** | Kelola dimensi P5 |
| **Dapodik** | Sinkronisasi data |
