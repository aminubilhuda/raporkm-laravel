# Prakerin

Panduan mengelola Praktik Kerja Lapangan (PKL) / Prakerin.

## Tempat Prakerin

### Daftar Tempat

1. Klik menu **Prakerin** di sidebar
2. Anda akan melihat daftar tempat prakerin

### Tambah Tempat Prakerin

1. Klik **Tambah Tempat**
2. Isi data:
    - **Nama Perusahaan**: Nama tempat PKL
    - **Alamat**: Alamat perusahaan
    - **Kontak**: Nomor telepon/email
    - **Keterangan**: Info tambahan
3. Klik **Simpan**

### Edit/Hapus Tempat

- Klik ikon **Edit** untuk mengubah
- Klik ikon **Trash** untuk menghapus

## Peserta Prakerin

### Daftar Peserta

1. Klik menu **Prakerin** → **Peserta**
2. Anda akan melihat daftar siswa yang mengikuti PKL

### Tambah Peserta

1. Klik **Tambah Peserta**
2. Isi data:
    - **Siswa**: Pilih siswa
    - **Tempat Prakerin**: Pilih perusahaan
    - **Pembimbing**: Pilih guru pembimbing
    - **Tahun Pelajaran**: Pilih TP
    - **Semester**: Pilih semester
3. Klik **Simpan**

### Import Peserta

1. Klik **Import** di halaman Peserta Prakerin
2. Pilih file CSV/Excel
3. Klik **Import**

## Pembimbing Prakerin

- Saat menambah peserta, pilih guru sebagai **pembimbing**
- Guru pembimbing akan melihat menu **Nilai Prakerin** dan **Rapor PKL** di panel mereka
