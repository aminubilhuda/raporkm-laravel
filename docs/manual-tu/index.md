## Hak Akses

Sebagai operator TU, Anda memiliki akses penuh ke seluruh fitur admin:

- Kelola data sekolah, pengguna, siswa, dan guru
- Setup kurikulum (mapel, tingkat, kompetensi)
- Kelola P5, kokurikuler, ekstra, prakerin
- Cetak rapor dan ekspor data
- Sinkronisasi dengan Dapodik
