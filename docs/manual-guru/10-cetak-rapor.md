# Cetak Rapor

Panduan cetak rapor digital (PDF) per siswa atau batch.

## Daftar Siswa

1. Klik menu **Cetak Rapor** di sidebar
2. Pilih kelas
3. Anda akan melihat daftar siswa di kelas tersebut

## Pilih Jenis Rapor

| Jenis | Keterangan |
|-------|-----------|
| **Semester** | Rapor akhir semester |
| **Mid** | Rapor tengah semester |

## Cetak per Siswa

1. Centang siswa yang ingin dicetak
2. Pilih jenis rapor (Semester/Mid)
3. Klik **Cetak PDF**
4. File PDF akan otomatis terdownload

## Cetak Batch (ZIP)

1. Centang beberapa siswa (minimal 2)
2. Pilih jenis rapor
3. Klik **Cetak Batch**
4. File ZIP akan terdownload berisi PDF per siswa

## Isi Rapor

Rapor yang dihasilkan berisi:

- **Cover**: Identitas sekolah dan siswa
- **Identitas Siswa**: Nama, NIS, NISN, kelas
- **Nilai Akademik**: Nilai per mata pelajaran
- **Nilai P5**: Penilaian sub-elemen P5
- **Nilai Kokurikuler**: Penilaian kokurikuler
- **Presensi**: Rekap kehadiran
- **Catatan Wali Kelas**: Deskripsi perkembangan
- **Tanda Tangan**: Kepala sekolah dan wali kelas

## Tanda Tangan Digital (Kepsek)

1. Login sebagai **Kepala Sekolah**
2. Buka menu **Rapor** di panel guru
3. Pilih siswa
4. Rapor akan menampilkan tanda tangan digital
5. Klik **Cetak** untuk mencetak rapor yang sudah ditandatangani

!!! tip "Tips Cetak"
    - Gunakan Chrome untuk hasil PDF terbaik
    - Pastikan koneksi internet stabil untuk cetak batch
    - Cek preview sebelum mencetak
    - Gunakan kertas A4 untuk hasil optimal

!!! warning "Penting"
    - Pastikan semua nilai sudah diinput sebelum cetak rapor
    - Pastikan catatan wali kelas sudah lengkap
    - Cek presensi siswa sudah lengkap
