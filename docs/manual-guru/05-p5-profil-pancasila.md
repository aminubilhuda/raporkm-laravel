# P5 & Profil Pancasila

Panduan mengelola project P5 dan penilaian sub-elemen.

## Project Kelas (P5)

### Daftar Project

1. Klik menu **Project Kelas** di sidebar
2. Pilih kelas wali
3. Anda akan melihat daftar project P5

### Tambah Project

1. Pilih kelas wali
2. Klik **Tambah Project**
3. Isi data:
    - **Tema**: Pilih tema P5
    - **Judul**: Judul project
    - **Deskripsi**: Deskripsi singkat (opsional)
4. Klik **Simpan**

### Hapus Project

1. Di daftar project, klik ikon **Trash**
2. Konfirmasi penghapusan

## Penilaian Sub-Elemen

### Buka Form Penilaian

1. Di daftar project, klik tombol **Nilai**
2. Anda akan melihat grid penilaian:
    - **Baris**: Siswa
    - **Kolom**: Sub-elemen per dimensi P5
3. Isi nilai untuk setiap siswa per sub-elemen (0-100)

### Simpan Nilai

1. Isi semua nilai yang diperlukan
2. Klik **Simpan** di bagian bawah form
3. Nilai akan tersimpan ke database

!!! tip "Tips Penilaian P5"
    - Nilai sub-elemen harus antara 0-100
    - Nilai kosong akan diabaikan
    - Anda bisa menyimpan sebagian nilai terlebih dahulu
    - Nilai yang sudah tersimpan akan muncul saat membuka form lagi

## P5BK (View)

1. Klik menu **P5BK** di sidebar
2. Anda akan melihat ringkasan project P5 yang diampu
3. Klik link project untuk melihat detail penilaian
