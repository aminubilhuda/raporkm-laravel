# Penilaian Akademik

Panduan input dan pengolahan nilai siswa.

## Tujuan Pembelajaran (TP)

### Daftar TP

1. Klik menu **Tujuan Pembelajaran** di sidebar
2. Pilih kelas dan mapel
3. Anda akan melihat daftar TP yang sudah dibuat

### Tambah TP

1. Pilih kelas dan mapel
2. Klik **Tambah TP**
3. Isi data:
    - **Kode TP**: Kode unik
    - **Deskripsi**: Uraian tujuan pembelajaran
    - **Elemen**: Pilih elemen dari P5
4. Klik **Simpan**

### Edit/Hapus TP

- Klik ikon **Edit** untuk mengubah
- Klik ikon **Trash** untuk menghapus

## Input Nilai

### Formatif

1. Klik menu **Penilaian** di sidebar
2. Pilih kelas dan mapel
3. Di bagian **Formatif**, isi nilai untuk setiap siswa
4. Klik **Simpan**

### Sumatif — Proyek (PH)

1. Di bagian **Sumatif PH**, isi nilai proyek
2. Nilai PH akan terhitung otomatis

### Sumatif — Tes Akhir (TS)

1. Di bagian **Sumatif TS**, isi nilai tes akhir
2. Nilai TS akan terhitung otomatis

### Sumatif — Asesmen Sumatif (AS)

1. Di bagian **Sumatif AS**, isi nilai asesmen sumatif
2. Nilai AS akan terhitung otomatis

!!! tip "Tips Input Nilai"
    - Gunakan **Tab** untuk berpindah antar kolom
    - Nilai hanya boleh antara 0-100
    - Nilai kosong akan diabaikan
    - Gunakan **Batch Update** untuk input cepat

## Lager Nilai Kelas

1. Klik menu **Lager Nilai** di sidebar
2. Pilih kelas
3. Anda akan melihat rekap nilai seluruh siswa di kelas tersebut
4. Klik **Export PDF** untuk mengunduh lager nilai

## Export PDF Lager

1. Pilih kelas
2. Klik **Export PDF**
3. File PDF akan terdownload berisi rekap nilai per siswa
