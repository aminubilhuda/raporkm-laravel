# Ekstrakurikuler & Presensi

Panduan mengelola ekstrakurikuler, presensi, dan absensi BK.

## Ekstrakurikuler

### Daftar Eskul

1. Klik menu **Ekstra** di sidebar
2. Anda akan melihat daftar eskul yang dibina
3. Informasi yang ditampilkan:
    - Nama eskul
    - Jumlah anggota
    - Jadwal kegiatan

!!! info "Info"
    - Menu eskul hanya muncul jika Anda terdaftar sebagai **Pembina Eskul**
    - Hubungi TU untuk mendaftar sebagai pembina

## Presensi

### Input Presensi Harian

1. Klik menu **Presensi** di sidebar
2. Pilih kelas
3. Untuk setiap siswa, tandai status:
    - **H**: Hadir
    - **I**: Izin
    - **S**: Sakit
    - **A**: Tanpa Keterangan
4. Isi keterangan (opsional)
5. Klik **Simpan**

### Rekap Presensi

1. Klik menu **Rekap Presensi** di sidebar
2. Pilih kelas
3. Anda akan melihat rekap presensi per siswa:
    - Jumlah hadir
    - Jumlah izin
    - Jumlah sakit
    - Jumlah alpha
    - Persentase kehadiran

## Absensi BK

### Daftar Absensi

1. Klik menu **Absensi BK** di sidebar
2. Anda akan melihat daftar siswa yang melapor ke BK

### Input Absensi BK

1. Klik **Tambah Absensi**
2. Isi data:
    - **Siswa**: Pilih siswa
    - **Tanggal**: Tanggal absensi
    - **Keterangan**: Alasan absensi
    - **Tindakan**: Tindakan yang diambil
3. Klik **Simpan**

!!! tip "Tips Presensi"
    - Input presensi setiap hari sebelum pelajaran selesai
    - Rekap presensi akan otomatis terhitung
    - Data presensi akan muncul di rapor siswa
