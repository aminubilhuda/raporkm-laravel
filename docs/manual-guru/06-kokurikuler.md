# Kokurikuler

Panduan mengelola kokurikuler dan penilaiannya.

## Dimensi Kokurikuler

1. Klik menu **Kokurikuler** di sidebar
2. Anda akan melihat daftar dimensi kokurikuler
3. Informasi yang ditampilkan:
    - Nama dimensi
    - Keterangan
    - Urutan

## Deskripsi Kokurikuler

1. Di halaman kokurikuler, Anda akan melihat deskripsi per dimensi
2. Deskripsi dibuat oleh Tata Usaha

## Penilaian Kokurikuler

### Daftar Penilaian

1. Klik menu **Penilaian Kokurikuler** di sidebar
2. Pilih kelas
3. Anda akan melihat daftar siswa

### Input Nilai

1. Pilih kelas
2. Untuk setiap siswa, isi nilai kokurikuler
3. Nilai berupa deskripsi/teks (bukan angka)
4. Klik **Simpan**

!!! info "Info"
    - Penilaian kokurikuler berupa deskripsi perkembangan siswa
    - Nilai akan muncul di rapor kokurikuler
    - Isi deskripsi yang objektif dan konstruktif
