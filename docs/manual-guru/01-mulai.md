# Mulai

Panduan login dan penggunaan dashboard guru.

## Login

1. Buka browser dan akses `https://km.smkabdinegara.sch.id`
2. Masukkan **username** dan **password**
3. Klik **Log in**
4. Anda akan diarahkan ke **Dashboard Guru**

## Dashboard

Dashboard menampilkan:

- **Statistik**: Jumlah kelas yang diampu, siswa, nilai terakhir
- **Aktivitas Terakhir**: Input nilai terbaru
- **Pengumuman**: Info dari admin/tata usaha

## Pilih Semester

1. Di sidebar, gunakan **Semester Switcher** untuk memilih tahun ajaran dan semester
2. Data akan otomatis berubah sesuai semester yang dipilih

!!! tip "Tips"
    - Pastikan semester yang dipilih sudah benar sebelum input nilai
    - Data nilai akan terkait dengan semester yang aktif
