# Prakerin

Panduan mengelola bimbingan prakerin, nilai, dan rapor PKL.

## Daftar Bimbingan

1. Klik menu **Prakerin** di sidebar
2. Anda akan melihat daftar siswa yang dibimbing
3. Informasi yang ditampilkan:
    - Nama siswa
    - Tempat prakerin
    - Status bimbingan

## Nilai Prakerin

### Daftar Nilai

1. Klik menu **Nilai Prakerin** di sidebar
2. Anda akan melihat daftar siswa yang dinilai

### Input Nilai

1. Klik **Edit** di sebelah siswa
2. Isi nilai per mata pelajaran prakerin:
    - Pilih mata pelajaran
    - Isi nilai (0-100)
3. Klik **Simpan**

### Validasi Nilai

- Nilai hanya boleh antara 0-100
- Nilai kosong akan ditandai sebagai belum dinilai
- Nilai yang sudah tersimpan bisa diubah kapan saja

## Rapor PKL

### Daftar Rapor

1. Klik menu **Rapor PKL** di sidebar
2. Anda akan melihat daftar siswa yang sudah dinilai

### Cetak Rapor PKL

1. Pilih siswa
2. Klik **Cetak PDF**
3. File PDF rapor PKL akan terdownload

!!! info "Info"
    - Rapor PKL berisi:
      - Identitas siswa
      - Data tempat prakerin
      - Nilai per mata pelajaran
      - Deskripsi perkembangan
      - Tanda tangan pembimbing
