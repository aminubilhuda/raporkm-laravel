## Panel Guru

Sebagai guru, Anda memiliki akses ke:

- **Dashboard**: Ringkasan kelas dan aktivitas
- **Penilaian**: Input dan kelola nilai siswa
- **P5**: Kelola project dan penilaian P5
- **Presensi**: Input presensi harian
- **Cetak Rapor**: Cetak rapor PDF

!!! info "Sidebar Dinamis"
    Menu di sidebar akan muncul secara otomatis berdasarkan peran Anda:
    - **Wali Kelas**: Menu catatan rapor, cetak rapor, project kelas
    - **Pengajar**: Menu penilaian, lager nilai
    - **Pembina Eskul**: Menu ekstrakurikuler
    - **Pembimbing PKL**: Menu prakerin, nilai prakerin
