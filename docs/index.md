# E-Rapor KM — Panduan Penggunaan

Selamat datang di **E-Rapor KM**, aplikasi rapor berbasis web untuk Kurikulum Merdeka.

## Tentang Aplikasi

E-Rapor KM adalah sistem informasi rapor digital yang membantu sekolah dalam:

- Mengelola data siswa, guru, dan kelas
- Input dan pengolahan nilai Kurikulum Merdeka
- Cetak rapor digital (PDF)
- Integrasi dengan Dapodik
- P5 (Projek Penguatan Profil Pelajar Pancasila)

## Panel Pengguna

| Panel | Pengguna | Akses |
|-------|----------|-------|
| **Tata Usaha (TU)** | Operator sekolah | `/tu/*` |
| **Guru** | Guru mapel, wali kelas, pembina | `/guru/*` |
